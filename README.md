- logo.jpg

## How to host on GitHub Pages
1. Upload all files to your GitHub repository root.
2. Open repository **Settings**.
3. Open **Pages**.
4. In **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save.
6. Wait 1–2 minutes and open your GitHub Pages link.
